# Singapore Job Market Insights Dashboard

This Streamlit dashboard helps:

**Employers**
- Benchmark salary by category and position level
- Understand hiring demand trends
- Gauge market competition using views vs applications

**Jobseekers**
- Identify higher paying categories and roles
- Prioritise applications in less competitive segments

## How to Run

```bash
pip install -r requirements.txt
streamlit run app/app.py
